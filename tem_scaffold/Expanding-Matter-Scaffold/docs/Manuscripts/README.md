# Manuscripts

Put your canonical papers/chapters here, e.g.
- `TEM-00_Existential_Necessity_of_Expansion.md`
- `TEM-01_Rational_Resonance_Radix.md`
- `TEM-02_Projective_Completeness_Theorem.md`

**Conventions**
- Place long proofs in `../Proofs/` and link to them.
- Use relative links: `../Proofs/Proofs-Only.md`
- Keep figures in `../Figures/` and link with `![](../Figures/diagram.png)`.

**Template**
See `../Templates/TEMPLATE_Paper.md` for a ready-to-copy manuscript skeleton.
